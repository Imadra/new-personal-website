ABCHack 2017

The project was supposed to make an automated video interview. Many participants made it by using ffmpeg library in Android in order to make merges of the videos and putting a question into the videos. However, we made it in a cleverer and time-saving way. There was no use of ffmpeg, we completed the app just by screencasting the videos and putting questions into the screencasted video. This makes the process 60 times faster then ffmpeg. Skills used: Java, Android Studio.

HackNU 2018

Implemented a website questionnaire. In order to participate in questionnaire, user should register via social networks. Main purpose was to get more information about a user through the social networks API. My project also used third-party servises in order to know about a user more

ABCHack 2018

Project was designed to group the students with the courses they are taking. When user types the acronym of the course they are taking in the semester, bot automatically adds the user into the group of that course. Also, in the group, by typing "sendfiles" command, bot sends a link to the google drive which has the materials of the course(syllabus, books). Skills used: Ruby, Node.js, Telegram-cli. Username of a bot: @CourseGroupBot
